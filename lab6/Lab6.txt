{</p><p>    try {</p><p>      const response = await axios.get(</p><p>        'http://localhost:3001/users'</p><p>      );</p><p>      const users = response.data;</p><p>      const user = users.find(</p><p>        (u) =&gt; u.username === credentials.username</p><p>          &amp;&amp; u.password === credentials.password</p><p>      );</p><p>      if (user) {</p><p>        const { password, ...safeUser } = user;</p><p>        return safeUser;  // → payload ở fulfilled</p><p>      } else {</p><p>        return rejectWithValue(</p><p>          'Tài khoản hoặc mật khẩu không chính xác'</p><p>        );</p><p>      }</p><p>    } catch (error) {</p><p>      return rejectWithValue(</p><p>        'Không thể kết nối đến máy chủ'</p><p>      );</p><p>    }</p><p>  }</p><p>);</p><p><strong>Xử lý 3 states trong slice bằng extraReducers:</strong></p><p>const authSlice = createSlice({</p><p>  name: 'auth',</p><p>  initialState,</p><p>  reducers: {</p><p>    // Các action đồng bộ</p><p>    logout(state) { ... },</p><p>    clearError(state) { ... },</p><p>  },</p><p>  // extraReducers — xử lý actions từ createAsyncThunk</p><p>  extraReducers: (builder) =&gt; {</p><p>    builder</p><p>      // Khi API bắt đầu (= LOGIN_START)</p><p>      .addCase(loginThunk.pending, (state) =&gt; {</p><p>        state.loading = true;</p><p>        state.error = null;</p><p>      })</p><p>      // Khi API thành công (= LOGIN_SUCCESS)</p><p>      .addCase(loginThunk.fulfilled, (state, action) =&gt; {</p><p>        state.loading = false;</p><p>        state.error = action.payload;  // error msg</p><p>      });</p><p>  },</p><p>});</p><p><strong>Sử dụng trong LoginForm:</strong></p><p>// CŨ:</p><p>const result = await loginAction(dispatch,</p><p>  { username, password });</p><p>if (result.success) { ... }</p><p>// MỚI:</p><p>import { loginThunk } from '../redux/slices/authSlice';</p><p>const result = await dispatch(</p><p>  loginThunk({ username, password })</p><p>);</p><p>// Kiểm tra kết quả:</p><p>if (loginThunk.fulfilled.match(result)) {</p><p>  // Thành công → navigate</p><p>  navigate('/');</p><p>} else {</p><p>  // Thất bại → error đã tự lưu vào state.auth.error</p><p>}</p><p><strong>Luồng hoàn chỉnh khi user click Login:</strong></p><p>User click Login</p><p>    │</p><p>    ▼</p><p>dispatch(loginThunk({ username, password }))</p><p>    │</p><p>    ├──→ Redux tự dispatch: {type:'auth/login/pending'}</p><p>    │       → state.loading = true (hiện spinner)</p><p>    │</p><p>    ├──→ Gọi API: axios.get('http://localhost:3001/users')</p><p>    │</p><p>    ├── Thành công?</p><p>    │   ├── Có user → return safeUser</p><p>    │   │   → dispatch {type:'auth/login/fulfilled'}</p><p>    │   │       → state.user = safeUser ✓</p><p>    │   │</p><p>    │   └── Không tìm → rejectWithValue('Sai MK')</p><p>    │       → dispatch {type:'auth/login/rejected'}</p><p>    │           → state.error = 'Sai MK'</p><p>    │</p><p>    └── Network error</p><p>        → dispatch {type:'auth/login/rejected'}</p><p>            → state.error = 'Không kết nối được'</p><h1><strong>PHẦN III — IMMER: TẠI SAO ĐƯỢC "MUTATE" STATE?</strong></h1><h3><strong>Nguyên tắc bình thường trong React:</strong></h3><p>// SAI — mutate trực tiếp:</p><p>state.loading = true;</p><p>return state;</p><p>// ĐÚNG — tạo object mới:</p><p>return { ...state, loading: true };</p><h3><strong>Trong Redux Toolkit, bạn ĐƯỢC viết:</strong></h3><p>loginStart(state) {</p><p>  state.loading = true;
